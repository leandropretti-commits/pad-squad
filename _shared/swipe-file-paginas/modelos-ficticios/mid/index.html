Não. O método foi pensado pra confeiteira de cozinha comum. Forno doméstico, batedeira normal. O que entrega é o posicionamento, não o maquinário.

Todas as aulas são gravadas pra você assistir no seu ritmo. A novidade é a Call Mensal comigo (bônus) — essa é ao vivo, 1x no mês, e fica gravada pra quem não puder.
