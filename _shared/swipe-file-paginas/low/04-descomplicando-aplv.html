Veredicto
Identidade visual forte em cores (roxo/rosa), mas precisa consolidar tipografia e criar tokens proprios.
Recomendacoes Prioritarias
1. Reduzir familias tipograficas. 6 fontes e excessivo. Consolidar em Poppins (headings/UI) + Merriweather (corpo editorial) + Bebas Neue (display/numeros). Remover Roboto, Montserrat e system stack.
2. Implementar design tokens proprios. Atualmente usa apenas CSS variables padroes do WordPress. Adotar as variaveis --aplv-* recomendadas acima para garantir consistencia.
3. Padronizar botoes. CTA e tabs/accordions tem estilos completamente diferentes (radius, cores, pesos). Criar uma escala de botoes coesa mantendo a hierarquia visual.
4. Expandir uso de border-radius. Apenas 1 valor (8px) encontrado. Definir escala de radius (4px, 8px, 16px, pill) e aplicar consistentemente em cards, botoes e inputs.
5. Normalizar escala de espacamento. O salto de 30px para 200px quebra a escala. Adotar multiplos de 4px ou 8px (4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80).
